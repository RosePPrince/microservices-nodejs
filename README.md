# microservices-nodejs

Design	and	implement	a	simple	microservices-based	system	using	Node.js.	The	system	will	
consist	of	two	microservices:	User	Service and	Product	Service.	Each	service	should	operate	
independently	and	communicate	as	necessary.	The	API	endpoints	must	be	documented	
using	OpenAPI	(Swagger),	allowing	for	the	endpoints	to	be	exercised	and	verified

## Setup Instructions

Follow these steps to set up your development environment:

1. **Clone the Repository**

   git clone https://github.com/RosePPrince/microservices-nodejs
   cd your-repository
   
2. **Install Dependencies**

   npm install
   
4. **Set Up Environment Variables**

   Create a .env file in the root of the project and add the necessary environment variables.Update the .env file with your specific configuration.

5.**Start the Development Server**
    npm start
    Run the Application in port specified in your configuration

6.**Run Unit Tests**
    npm test
